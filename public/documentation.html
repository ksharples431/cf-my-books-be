My Books API: Documentation

About this API

A simple API that provides users with information about books, authors, and genres.

The Api's features include

  • Allows new users to register
  • Allows users to login
  • Returns list of all users
  • Returns data about a single user
  • Allows users to update thier user info
  • Allows existing users to deregister
  • Allows users to add a book to a list of their favorites
  • Allows users to remove a book from their favorites list
  • Returns list of all books
  • Returns data about any single book including description, author, genre
  • Returns data about a specific author
  • Returns data about a specific genre
  • Allows users to add a book
  • Allows users to edit a book

Endpoints

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Allows new users to register /signup POST - A JSON object holding data about the user to register, structured as: { username: req.body.username }. Including - email, password, birthday. A JSON object holding data about the user that was added
Allows a user to login /login POST - A JSON object holding data about the user to login, structured as: { username: req.body.username }. Including - email, password, birthday. A JSON object holding data about the user that was added
Returns a list of all users /users GET - - A JSON object holding data about all the users
Returns data about a single user by user id /users/:uid GET :uid - A JSON object holding data about a single user
Allows users to update their user info /users/:uid PATCH :uid A JSON object holding data with the user's updated info, structured as: { username: req.body.username }. Including - email, password, birthday A JSON object holding data about the updated user information
Allows existing users to deregister /users/:uid DELETE :uid - A text message indicating whether the user has successfully deregistered
Allows users to add a book to their favorites /users/:uid/:bid POST :uid, :bid - A JSON object holding data about the user and the book that was added
Allows users to remove a book from their favorites /users/:uid/:bid DELETE :uid, :bid - A text message indicating whether the user has successfully removed the favorite
Returns a list of all books /books GET - - A JSON object holding data about all the books
Returns data about a single book by title /books/:title GET :title - A JSON object holding data about a single book
Returns data about an author by author name /books/:authorName GET :authorName - A JSON object holding data about a specific author
Returns data about a genre by genre name /books/:genreName GET :genreName - A JSON object holding data about a specific genre of book
Allows a user to add a book /books/add POST - A JSON object holding data about the book, structured as: { title: req.body.title }. Including - title, author, image, description, genre, series, and seriesNumber A JSON object holding data about the book that was added
Allows a user to edit a book /books/:bid PATCH :bid A JSON object holding data about the book, structured as: { title: req.body.title }. Including - title, author, image, description, genre, series, and seriesNumber A JSON object holding data about the book that was added
